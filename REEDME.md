# 契約書管理システム (Contract Manager)

## 概要
本プロジェクトは、管理者とユーザー（消費者）間での契約書のライフサイクル管理、合意形成、および証拠保全を行うための高度なセキュリティを備えたWPFアプリケーションです。重要契約においては、確実な同意取得のための音声録音と、本人確認書類の安全な暗号化保存を必須としています。

## 主な機能
- **アカウント管理**: PBKDF2アルゴリズムを用いた安全な認証システム。初回起動時にのみ管理者セットアップ画面を表示し、ハードコードされた認証の抜け道を防止。
- **契約の割当とステータス追跡**: 契約書に対する同意、拒否、異議申し立て、代理同意などを個別にステータスとして管理。
- **重要契約における高度な同意プロセス**: 
  - 事前説明（HTMLによるリッチテキスト表示）の義務化。
  - 事前説明閲覧中のPCマイクによる自動音声録音。録音データはSHA-256でハッシュ化し改ざんを防止。
  - ECC（楕円曲線暗号 NIST P-256）による鍵交換とAES-256-GCMによるハイブリッド暗号化を用いた、本人確認書類画像のセキュアなアップロードと保存機能。
- **安全な監査・復号機能**: 管理者ダッシュボードの専用UIから、暗号化されたデータの復号および音声データの再生による検証が可能。

## インストールおよびビルド手順
本システムは .NET 8.0 をベースとしています。

1. **前提条件**
   - Windows 10 または Windows 11
   - .NET 8.0 SDK のインストール

2. **ビルド**
   プロジェクトディレクトリの `契約書システム-src` フォルダでコマンドラインを開き、以下のコマンドを実行してください。
   ```bash
   dotnet build ContractManager.csproj
   ```

3. **実行**
   ```bash
   dotnet run --project ContractManager.csproj
   ```

## 技術スタック
- **UIフレームワーク**: Windows Presentation Foundation (WPF) / XAML
- **言語**: C# 12 / .NET 8.0
- **データ永続化**: JSON ベースのローカルファイルシステムストア (`System.Text.Json` 8.0.4 適用済み)
- **暗号・セキュリティライブラリ**: `System.Security.Cryptography`
